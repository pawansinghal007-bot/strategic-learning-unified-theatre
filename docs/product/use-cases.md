# Use Cases

## Local continuity for AI-assisted work
A developer uses online LLMs and VS Code together, with the system preserving context so the next session starts with relevant memory.

## Structured learning from AI interactions
Browser responses and editor signals are captured, ingested, and made available for prompt generation rather than being lost in chat history.

## Local-first privacy-sensitive workflows
Users who prefer local storage can keep experience data on their machine and avoid unnecessary cloud persistence for current project state.

## Sprint handoff and follow-up
Handoff snapshots and curated docs make it easier to resume work without reconstructing state from fragmented notes.
