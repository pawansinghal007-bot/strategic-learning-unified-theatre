# strategic-learning-unified-theatre — Product Overview

## Summary
strategic-learning-unified-theatre is a local-first development intelligence product that captures learning signals from real AI-assisted work and turns them into better prompts, reusable memory, and more reliable future sessions.

## Problem
Developers lose context between AI sessions.
Useful prompts, decisions, mistakes, and outputs are scattered across chats, files, editors, and browser sessions.
This creates repeated work, inconsistent outputs, and weak institutional memory.

## Solution
The product creates a persistent learning layer across developer workflows.
It captures relevant signals from browser-based LLM use, VS Code activity, and local project history, then stores them in a structured experience database.
That database is used to improve future prompts, preserve continuity, and support more consistent follow-up work.

## Core Capabilities
- Browser-to-experience capture
- VS Code passive learning
- Experience database with retrieval
- Mistake tracking and rubric promotion
- Prompt generation using prior context
- Handoff continuity across sessions
- Training export readiness for future local fine-tuning workflows

## Who It Is For
- Solo developers using multiple LLMs
- Technical founders building with AI assistance
- Research-heavy engineering teams
- Power users who want local memory and continuity
- Privacy-conscious users who prefer local-first workflows

## Value Proposition
The product helps users keep context, reduce repeated prompting, improve output quality, and build a compounding memory system from daily AI work.

## Differentiators
- Local-first workflow
- Cross-session continuity
- Cross-tool memory capture
- Structured experience database
- Fine-tuning readiness path
- Audit-friendly operational history

## Current Scope
Current scope includes browser capture, VS Code capture, experience ingestion, retrieval, handoff support, and export readiness for training data workflows.

## Not Yet in Scope
- Production-grade team admin
- Enterprise permission model
- Hosted cloud control plane
- Mature analytics dashboards
- Full customer-facing documentation suite

## Example Use Case
A developer asks ChatGPT for a refactor, tests the change in VS Code, gets diagnostics, fixes the issue, and closes the sprint.
The product preserves that trail so the next session starts with the right context instead of rebuilding everything from memory.

## Status
The system is in active development with core capture, storage, and prompt-improvement workflows already implemented.
