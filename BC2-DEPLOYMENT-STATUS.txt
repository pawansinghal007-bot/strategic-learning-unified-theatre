Step 1: Load the Browser Extension
─────────────────────────────────────────────────────────────────────────────
Firefox:
  1. Open Firefox
  2. Type: about:debugging
  3. Click: "This Firefox" on the left
  4. Click: "Load Temporary Add-on"
  5. Select: C:\SW Development\VS Code Agent\files\bc2-firefox\extension\firefox\manifest.json

Brave:
  1. Open Brave
  2. Type: brave://extensions
  3. Toggle: "Developer mode" (top right)
  4. Click: "Load unpacked"
  5. Select: C:\SW Development\VS Code Agent\files\bc2-brave\extension\brave
  6. Get Extension ID from the loaded extension
  7. Edit: C:\SW Development\VS Code Agent\files\bc2-native-host\native-host\brave_host_manifest.json
  8. Replace: "YOUR_BRAVE_EXTENSION_ID" with the actual ID
  9. Run: C:\SW Development\VS Code Agent\files\bc2-native-host\install.bat as Administrator

Step 2: Interact with Browser Extension
─────────────────────────────────────────────────────────────────────────────
1. Click the extension icon in your browser toolbar
2. Create an account or log in
3. Wait for native host connection confirmation

Step 3: Capture Some Chat Activity
─────────────────────────────────────────────────────────────────────────────
1. Navigate to: https://chat.openai.com or https://claude.ai
2. Send a test message: "Hello, this is a test"
3. Wait for the response to complete
4. Check the extension popup for capture status

Step 4: Verify Data Arrived
─────────────────────────────────────────────────────────────────────────────
Run any of these PowerShell commands:

# Check API health
curl http://localhost:7070/health

# View captured events
curl "http://localhost:7070/events?limit=5"

# Get event statistics
curl "http://localhost:7070/events/stats"

# Get formatted context for LLM
curl "http://localhost:7070/events/recent-context?n=5"

Step 5 (Optional): Test LLM Integration
─────────────────────────────────────────────────────────────────────────────
Start Ollama in a separate terminal:

Then test the /ollama/ask endpoint:
  curl -X POST http://localhost:7070/ollama/ask `
    -H "Content-Type: application/json" `
    -d '{"question": "What was I doing?", "context_n": 10}'

🔗 HOW THE PIPELINE WORKS
═══════════════════════════════════════════════════════════════════════════════

Browser Tab (ChatGPT / Claude / etc.)
        ↓
  Content Script (content.js)
        ↓ [chrome.runtime.sendMessage]
  Background Script (background.js)
        ↓ [chrome.runtime.connectNative]
  Native Host (host.py)
        ↓ [stdin/stdout]
  SQLite Database (capture.db)
        ↓
  Flask API (http://localhost:7070)
        ↓
  Your Application & LLM

💡 KEY FEATURES
═══════════════════════════════════════════════════════════════════════════════

✨ Passive Capture
  - Automatically captures messages from ChatGPT, Claude, Gemini, Perplexity
  - No manual copy-paste needed
  - Works while you're using the browser

✨ Local Storage
  - All data stored locally in SQLite
  - No cloud, no API keys, no subscriptions
  - Private and secure

✨ REST API
  - Query captured data via HTTP endpoints
  - Filter, search, and aggregate events
  - Export formatted context for LLM prompting

✨ LLM Integration
  - Send captured chat history to local Ollama
  - Get context-aware responses based on your interactions
  - Feedback loop improves future prompts

❓ TROUBLESHOOTING REFERENCE
═══════════════════════════════════════════════════════════════════════════════

Issue: API returns 500 errors
  → Check Flask API terminal logs
  → Ensure database schema is initialized
  → Run browser extension to trigger initialization

Issue: Extension doesn't load
  → Verify extension ID matches manifest
  → Check browser extension settings page
  → Review browser console (F12) for errors

Issue: No events captured
  → Verify extension is connected to native host
  → Check extension popup for status/errors
  → Send a message and wait for response to complete

Issue: Ollama endpoint fails
  → Start Ollama: ollama serve
  → Verify it's running: curl http://localhost:11434/api/tags
  → Install model: ollama pull llama3

Full troubleshooting guide: BC2-INTEGRATION-CHECKLIST.md
